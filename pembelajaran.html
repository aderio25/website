Jenis dan Bidang

Dalam komunitas ini, di awal perintisannya akan memiliki dua bidang utama antara lain :

  • Pemrograman Umum (Dianjurkan untuk pemula)

    • Berkaitan erat dengan struktur data dan algoritma juga bila memungkinkan pemrograman fungsional ataupun logika, luaran yang diharapkan yaitu memahami dengan baik konsep dasar pemrograman dan memiliki keterampilan problem solving terutama untuk menyelesaikan coding interview dan competitive programming

  • Pemrograman Terapan (Kurang dianjurkan untuk pemula)

    • Berkaitan erat dengan bidang pengembangan aplikasi web, mobile maupun gim, Internet of Things, Artificial Intelligence data sains, dan lainnya. Luaran yang diharapkan yaitu menghasilkan suatu proyek yang bermanfaat terutama bagi mahasiswa dan kampus


Bahasa pemrograman

Bahasa pemrograman bagi setiap anggota bersifat bebas, namun bagi pemula dianjurkan untuk menggunakan bahasa pemrograman utama yang akan digunakan di komunitas berikut :

  • Python
    Python bersifat serba guna di berbagai aspek
  • C++
    C++ cocok untuk memahami struktur data dan algoritma
  • JavaScript
    JavaScript cocok untuk pengembangan aplikasi website, mobile, hingga desktop
  • TypeScript
    TypeScript merupakan bahasa pemrograman website masa depan
  • Dart
    Dart merupakan bahasa pemrograman mobile masa depan
  • Clojure
    Clojure untuk memahami paradigma pemrograman dengan Functional Programming-nya
